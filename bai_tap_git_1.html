1. Git là gì?

- Git là một hệ thống điều khiển phiên bản theo hình thức phân tán được sử dụng phổ biến nhất.
- Git là một cách thức (mô hình, giao thức, công cụ, qui tắc) để lưu trữ mã nguồn,
chia sẻ mã nguồn, khôi phục mã nguồn, dễ dàng chỉnh sửa mã nguồn và hỗ trợ cộng tác trong quá trình phát triển ứng dụng.

2. Các câu lệnh cơ bản của git

- Git init: khởi tạo một local repo
- Git add (tên_file): đưa các file vào trong cùng staging area (có thể dùng '.' để đưa tất cả)
- Git commit -m "comment": đưa các file từ staging vào local repo
- Git remote add origin (đường_link): kết nối local repo với remote repo (github)
- Git push origin master: đẩy các thay đổi từ local repo lên remote repo
