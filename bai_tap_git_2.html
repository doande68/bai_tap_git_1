HTML là gì?

- HTML (Hypertext Markup Language) tạm dịch là ngôn ngữ đánh dấu siêu văn bản. HTML được sử dụng để tạo và cấu trúc các thành phần trong trang web hoặc ứng dụng,
phân chia các đoạn văn, heading, titles, blockquotes,… HTML không phải là một ngôn ngữ lập trình.

- Các trình duyệt web đọc và hiển thị mã HTML dể tạo ra giao diện người dùng tương ứng, HTML cung cấp cấu trúc cơ bản cho trang web và định dạng các phần tử trên trang web đó.
